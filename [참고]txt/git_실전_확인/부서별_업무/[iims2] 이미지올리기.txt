1. iims2에서 "새권한요청"버튼으로 권한 받을 디렉토리 설정 후
2. 상태 승인으로 뜨기를 기다린다. (승인여부중에는 볼 수는 있지만 데이터를 넣을 수는 없다.)
3. 그런후 선택에서 해당 서비스명 radio버튼 선택 후  "기본 접속 서비스 저장" 버튼 클릭 후 
4. FTP에 static 서버로 들어가면 해당 디렉토리로 들어가진다.
5. 넣어줘야 할 이미지 경로를 찾아서 파일을 넣은 후 iims2에서 "sync(싱크)" 버튼 클릭해서 완료 버튼이 뜨길 기디란다.
6. 해당 이미지 경로로 url로 확인하면 이미지가 나온다.


iims2에 넣은 이미지는 절대로 덮어씌우면 안된다. (운영반영되어 있는 이미지므로 덮어씌우기 금지임)
