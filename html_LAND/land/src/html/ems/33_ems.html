중개업소 회원정보와 첨부한 개설등록증의 정보가 일치하지 않으면
프로필 배너 신청이 반려됩니다.
아래 내용에 해당하지 않는지 살펴본 후 개설등록증을 다시 첨부하거나
회원정보를 바른 내용으로 수정해 주세요.
주요 반려 사유
- 개설등록증의 대표자와 기존 회원정보의 대표자가 일치하지 않은 경우
- 개설등록증의 정보(대표자명, 상호명, 소재지 등)를 식별할 수 없는 경우
- 회원정보와 개설등록증의 정보가 불일치하여 회원정보 수정이 필요한 경우
- 개설등록증이 첨부되지 않은 경우
